# Software

The rocket source code running on the raspberry pi can be [found here](https://github.com/noderockets/rocket).

## Edit Code
To edit the code directly on your pi, you'll need to ssh into your pi.

1. Open Terminal or a bash window
2. `$ ssh pi@10.0.0.84` (or whatever your sd card number is as the last number)
3. `cd rocket`
4. All the code is here
5. `sudo node rocket` to run the server
6. Go to `http://10.0.0.84`(or whatever your sd card number is as the last number) in a browser to run the mission control dashboard
