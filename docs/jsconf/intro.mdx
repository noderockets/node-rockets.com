# Intro

You will receive a kit. You can choose from a beginner or advanced kit. The advanced is the same but will require some soldering.

# Goal

Your mission is to assemble the electronics and build the rocket. Then create a strategy for detecting launch and deploying the parachute.

# Assembly Instructions

You can split up and work on some of these parts separately.

1. Advanced: Start with the [shield assembly](/js-conf/shield).
2. Beginner: Start with the [electronics assembly](/js-conf/electronics).
3. Then [rocket building](/js-conf/rocket).
